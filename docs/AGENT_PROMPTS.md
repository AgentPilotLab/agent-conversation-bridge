# Agent Prompt Examples

## Same Project Restart

```text
Use $agent-conversation-bridge to create a Bridge Packet for this repository so a new Codex chat can resume tomorrow.
```

## Local Project Bridge

```text
Use $agent-conversation-bridge to move this task into C:\path\to\target-project and create a receiving Codex thread if the tools are available.
```

## Machine Migration

```text
Use $agent-conversation-bridge to prepare this work for another computer. Include setup assumptions, missing credentials, validation status, and a starter prompt.
```

## Product Mechanism Clarification

```text
Use $agent-conversation-bridge to explain whether I need a Bridge Packet, Codex Remote hand off, or Import to Codex for this migration.
```
